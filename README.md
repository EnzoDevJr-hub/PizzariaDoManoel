# PizzariaDoManoel

## Status do Projeto
🚧 Em desenvolvimento

## Objetivo do Software
Sistema desktop de gerenciamento para a Pizzaria do Sr. Manoel, permitindo controle de pedidos, cadastro de clientes e movimentações de caixa de forma integrada.

## Tecnologias Utilizadas
- Java — Linguagem principal de desenvolvimento
- Java Swing — Interface gráfica desktop
- MySQL — Banco de dados relacional
- JDBC — Conexão Java com banco de dados
- Apache NetBeans — IDE de desenvolvimento

## Time de Desenvolvedores
- Enzo

## Funcionalidades do Sistema
- Login com autenticação de usuário e senha
- Cadastro de clientes com endereço completo
- Registro e finalização de pedidos com desconto
- Controle de caixa com entradas e saídas
- Cálculo automático de saldo
